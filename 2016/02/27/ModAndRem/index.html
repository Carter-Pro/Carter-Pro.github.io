数学上取模与取余

取模和取余的步骤

对于整数a和b来说，取模运算和取余运算步骤是一样的。

  1. 求整数商c

    $$
    c = a / b
    $$

    ​

  2. 计算模/余数r

    $$
    r = a - c * b
    $$

取模和取余区别

取模和取余的区别在于第一步计算整数商c时取整的方法不一样。

整数除法取整的3种方法

  1. 向上取整，向+∞方向取最接近精确值的整数，也就是取比实际结果稍大的最小整数，也叫 Ceiling 取整。这种取整方式下，17 / 10 == 2，5 / 2 == 3, -9 / 4 == -2。
  2. 向下取整，向-∞方向取最接近精确值的整数，也就是取比实际结果稍小的最大整数，也叫 Floor 取整。这种取整方式下，17 / 10 == 1，5 / 2 == 2, -9 / 4 == -3。
  3. 向零取整，向0方向取最接近精确值的整数，换言之就是舍去小数部分，因此又称截断取整（Truncate）。这种取整方式下，17 / 10 == 1，5 / 2 == 2, -9 / 4 == -2。

区别

其中，取余使用了向零取整，取模使用了向下取整。

因此，在被除数与除数符号一致的情况下，取模与取余结果是相同的。

计算机的取模和取余

在流行编程语言中（除了matlab），%就是取余（至少在C、C++、Java和C#是这样的），教科书将这两个概念同化了。
